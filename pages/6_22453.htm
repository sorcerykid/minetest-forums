Thanks, that was the one-line answer I needed! :D

For posterity:
To install X:
Code: Select all
brew cask install xquartz

To install the last 0.4.17.1 formula for minetest:
Code: Select all
brew install https://raw.github.com/Homebrew/homebrew-core/ffb065da3160a5fb65ae0bbd12e9a881168b25f9/Formula/minetest.rb

To pin the version so it doesn't get updated:
Code: Select all
brew pin minetest


Should work like a champ until Debian and Android all get updated.
