Build #20181201-0905: updated basic_materials, biome_lib, technic, digilines, digistuff, gloopblocks, homedecor, moreblocks, pipeworks, and prefab_redo

The Technic update in this build separates the CNC machine into its own mod: technic_cnc. Be sure you enable it in your world config, or you may end up with a bunch of unknown nodes.

Incidentally, as a result, the CNC no longer hard-depends on Technic, but will cooperate with it if enabled (and behave exactly as it always has), so you can enable it on worlds you otherwise won't use Technic on, if you just want the shapes it can make. In that situation, it has a different craft recipe, and will cut shapes instantly (similar to Moreblocks' table saw).

Re: [Modpack] Dreambuilder [20181217-0220][dreambuilder]

by VanessaE » Sun Dec 30, 2018 12:47 am

This is Minetest, we don't need crafting tables here. :-P Craft stairs by either a pattern like this:

X - -
X X -
X X X

...where X is the material you want to use, such as wood or stone (not all materials can make stairs etc). Several other shapes have recipes like that. Use the table saw ("circular saw") for more advanced shapes.
