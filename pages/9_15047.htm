[modpack] Modern ish [modern_ish]

by thomasthespacefox » Thu Jun 30, 2016 05:31

modern_ish is a modpack (formerly just a mod) that adds various decorative nodes meant to be used for decorations in creative mode, such as modular ceiling tiles.

Notably, it now has modern_ish_manholes, which provides working manholes & hatches, with an API to create them.


NOTE: this modpack currently has NO CRAFTING RECIPES. They are planned for a future release however.

you can get the mod here:
https://github.com/ThomasTheSpaceFox/minetest-modern_ish/releases

license: GNU GPL v3 (see also readme.txt file in mod's folder)

remember to rename the mod's folder to:
Code: Select all
modern_ish

if its not already that name.

tested with: Minetest v5.0.0

by thomasthespacefox » Tue Mar 19, 2019 21:12

v2.0 is here!

- Now a modpack
- new manholes & hatches with API (modern_ish_manholes)
- floor and wall drains in bronze, steel and gold.

Now depends on default (technically not yet, but just figured Id do that now rather than randomly through the 2.* version numbers.

Plans:
- add proper sounds
- add full set of crafting recipes
- add more optional components. such as doors, integrations with other mods, etc.
- add stairs & slabs for various blocks.

edit: v2.0.1:

I tried to make the lighting a bit more realistic. I wanted to make the more 'solid' manholes and hatches not propagate sunlight when closed, but due to possibly-inherit problems with minetest's lighting engine, i ended up having to disable sunlight propagation in the more 'solid' manholes and hatches, altogether.
