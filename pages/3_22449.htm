Re: [Poll] Which editor do you use for Lua code?

by micheal65536 » Fri Mar 29, 2019 9:52 pm

Pluma, the default text editor of the MATE desktop environment. Forked from Gedit back when GNOME 2 was still around and before the Gedit rewrite that made it practically useless as anything more than a note-taking application.

It has syntax highlighting and semi-automatic indentation. I actually use this for most of my code, except larger projects (especially Android) where keeping track of everything gets difficult and I can benefit from IDE features like refactoring, find definition/usages, and so on.

The one thing I miss though is code folding. There was a plugin that provided (somewhat hackish, but nevertheless I used it all the time) code folding in Pluma but the plugin stopped working when MATE switched to GTK 3 and I couldn't figure out how to update it to work again.
