You can always replace “hub” with “lab” :)

https://gitlab.com/4w/hunger_ng/issues/5

There is an option hunger_ng_use_hunger_bar you can set to false in minetest.conf. This prevents the hunger bar created by Hunger NG to be shown (anything else works like before, this option just hides the hunger bar and disables the functions to change the hunger bar) in case of other mods have issues with this.

Version 2.0.0!

by Wuzzy » Mon Apr 01, 2019 11:29 am

Version 2.0.0 is released that now supports Minetest 5.0.0!

  • Mod now depends on Minetest 5.0.0 or later
  • Fix stuff related to Minetest 5.0.0 breakage (translation, legacy functions, etc.)
  • Health and breath bar now support non-default maximum health and breath values
  • IMPORTANT: All mods that depend on this mod and have set a custom HUD bar format string (rather than using the default one) will break. Modders must update their mods accordingly, please refer to API.md. This step was required because of the new translation system. Note that most mods do not set the custom format string and will still work
