Re: How to setup on a hosted server?

by Linuxdirk » Sun Apr 07, 2019 03:45

EPROM1 wrote:1. On the minetest download page there are many versions, so which version should I download for my server?

Usually servers run on Linux and most Linux distributions have Minetest in their repositories. SO ideally you don’t manually load/install Minetest on servers but simply get the version from your server’s distribution’s repository. For distributions that only ship outdated packages for “stability reasons” custom repositories exist for example there is a PPA for Ubuntu.

EPROM1 wrote:2. I assume I would only require the minetest server?

Yes, you do not need the client when installing on a server. You need some of the Xorg libraries to compile it, though.

EPROM1 wrote:3. Is there a config file I could run on the server to instal minetest?

Don’t fall for the one-line script for “installing” self-compiled Minetest that is posted on the forums every now and then. For doing it properly just fetch the code and run ./configure. The build system will tell you what’s missing. Search the missing names via your distribution’s package manager’ search functionality.
