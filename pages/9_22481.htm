Pulverizer



Say NO to useless cobble, say NO to whatever (but not this mod)!

It allows you to pulverize any dug node.
Works with sfinv and default formspec.
(do not use it as storage for your item, you will lose your item on disconnect!)

If you dig stone, you will get different node. In this case, you will have to edit code and add aliases to table:
pulverizer.after_dig_nodes
Same with seeds, ores, etc.

Depends:
default
sfinv?
