by 1248 » Wed Apr 04, 2018 1:01 pm

Info:
Castrum is my new game for Minetest.
You are on a floor plan of a castle.
Hit a diamond block to build and upgrade a building.
Go first to the quarry in the east.
Collect resources and rebuild the old castle
Mods:
all minetest_game mods included
Mods from my team:
castrum by 1248
License:
See README.txt in each mod for more information
Every code written by me is LGPLv2.1 and CC-BY-SA
Download:
Note: If you update Castrum, you have to copy the players and SAVE folder from the old world into the new world folder. After this you have to run /update and you should get back your old castle.
