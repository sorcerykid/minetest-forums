Thank you, I added your textures.

A summary of recent changes:
  • Added Ruby Dust that generates in Ruby biome
  • Added Slime Bricks for all slime types
  • Implemented bucket support for slime
  • Implemented stairs, stairs redo and moreblocks support for some blocks
  • Mese Crystals now generate in Mese + Saphire biome
  • Dense ores and gems except quartz types are now 4 times less common
  • Luminosity of fiery blocks is reduced
  • Burning Mushroom and Underground Bush now use other textures
  • Added some crafting recipes
  • Crystals, moss and torchberries no longer generate in air in large caves.

link: https://forum.minetest.net/viewtopic.php?f=11&t=3958
