minetest_road:build_road a mod node that makes a stone brick road grid with 23x23 lots from where the node was placed and fixes itself.

See download attached.

Version: 1
License: GNU Affero General Public License version 3 or (at your option) any later version

Dependencies: default mod (found in minetest_game)

Report bugs or request help on the forum topic.

Installation



Unzip the archive, and place it in minetest/mods/minetest/

( GNU/Linux: If you use a system-wide installation place
it in ~/.minetest/mods/minetest/. )

( If you only want this to be used in a single world, place
the folder in worldmods/ in your worlddirectory. )

For further information or help see:
http://wiki.minetest.com/wiki/Installing_Mods
