[Mod] Guild / Party / Factions Mod [party]

by fishyWET » Wed Oct 04, 2017 12:54 pm

A simple mod that allow players to create and maintain their own parties & squads.
It functions much like factions plugin from minecraft, while also adding in some guild/party features from various other mmorpg/moba games.
Squads are subgroups of parties and adds an HUD display which shows fellow squad members' health.
In general, its main purpose is to function as a player management mod.

A great add-on for RPG/ team/ guild servers or for a multiplayer team game.


Image above shows the HUD display for squads feature.


NOTE: The mod hasn't been fully tested. Some errors and unexpected behavior are expected. Please report them over at the issues page or below.
Feedback is welcomed too.

Features include
  • Permanent status
  • Name your own party/ squad
  • No friendly-fire (Can be disabled individually)
  • Private party/ squad chat
  • Party nametag with colours
  • HUD health bars of squad members in squad mode
  • Give titles to your members
  • Invite and kick players
  • Promote players to officers to help manage your party
  • Many much more!
    For a full list, refer to the wiki


For a list of available commands, use "/p help" & "/sq help"
The wiki here also contains some information on how to use the commands.
For mod developers, the mod itself provides a way to handle user-made groups, which can be utilized for things like group healing etc. doc.txt provides information on how the storage works.

Download - Rename folder to "party"
Github

Compatibility: The mod modifies chat messages and nametags, any other mod that attempt to do so might not be compatible.

Update:


Couple of new features compared to the first release
  • New squad mode! Sub-groups of parties and can hold up to 7 members. Players in squads can see each other health bars on the left corner! Also adds a separate chat channel Credits to lisacvuk for idea
  • Couple of administrative things are added, admins can force join/ kick/ disband any party - also a spy mode that requires "pspy" privilege - all messages will be revealed to admins. Credits to lumberJ for idea
  • Add titles to players if you are the leader, with "/p title <playername> <title>" -- it makes their chat messages in party messages come up with an extra title!
  • If two players have "/p pvp" toggled on, it allows friendly fire.
  • Party home, "/p home" to teleport to a party home that the leader has set with "/p homeset"
  • Switch which chat you are talking on with ease now, using "/p chat <main/party/squad>". Additionally, adding "@" in front of your chat messages allow quick sending to the main channel no matter which chat mode you are in.
  • Couple of bugs squashed

For a full complete list, the commits should give an better idea.
Additionally, you could refer to the wiki.
For ingame help, use "/p help" and "/sq help"
