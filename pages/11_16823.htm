[Mod] Basic Announce System [announce]

by admicos » Wed Mar 01, 2017 20:53

I made this for my private server, but i think it will be useful for other people, so here it is!

This is a pretty basic mod that puts chat messages in a set interval, allowing you to create "announcements" and the like.

Repo: https://gitlab.com/admicos/minetest-announce
ContentDB: https://content.minetest.net/packages/admicos/announce/
License: MIT
Dependencies: none

Upcoming Features:
  • None so far, suggest one below!

The only file you need to edit is config.lua, color reference over at http://ess.khhq.net/mc/ (Not my site, but the coloring system is the same)
