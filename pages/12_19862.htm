Simple Texture-Pack Test Map

by MysticTempest » Fri Mar 23, 2018 2:52 am

So, I needed a texture pack test map for Minetest Game to fully display my textures.
But, there doesn't seem to be any current ones; so I went ahead and created a simplistic one.

I'm pretty sure I got all the major nodes with unique textures, but it's possible I missed something.
The world is creative, and 'flat'; with a simple tower platform to stand on for the perfect view.

It is to be used with "Minetest 0.5.0-dev". As I've added in the new nodes like the kelp node.
So, just a heads up if you use another version; as you will see unknown nodes.

Anyway, hope this is of some use to any other texture pack creators.

Edit: Update - October 2018 - Added berry bush leaves.
