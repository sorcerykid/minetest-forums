[Mod] Animachines (WIP)

by oscar14 » Thu Mar 07, 2019 7:49 pm

I don't use mobs in my worlds but I still need a way to get some items, thats why I'm making this mod.

This mod add "machines" that give you some animal items (meet, leather, etc).



https://github.com/camus14/animachines
Download: https://github.com/camus14/animachines/archive/master.zip
Licence: MIT License.

Notes:
- It doesn't have recipe yet.
- I want to add more animachines (chicken, pig...).

Update:
- Added a new texture for the node.
- Now you can see the satiation progress.
