Are there these entity drawing settings?

by Hume2 » Thu Mar 14, 2019 06:29

In my Wings of Escape mod, I'd like to use these effects for my entity. Firstly, I'd like to make the brightness independent on the brightness of the environment. It should be always drawn in full brightness, even in the dark. Secondly, I'd like to use alpha channel for my textures. Currently, each pixel can be either opaque or transparent, nothing in between.

Is there a way to make these effects?

Re: Are there these entity drawing settings?

by ratmix » Thu Mar 14, 2019 23:10

Use this for entity transparency: use_texture_alpha = true
https://github.com/minetest/minetest/blob/master/doc/lua_api.txt#L5592


And this for glow: glow = <value>
https://github.com/minetest/minetest/blob/master/doc/lua_api.txt#L5630
