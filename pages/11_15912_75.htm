Version 1.3.0!

by Wuzzy » Tue Mar 19, 2019 6:15 am

Version 1.3.0 is out!
This is mostly a maintenance update for MT 5.0.0.

Changelog:
  • All mods: Switch to new translation system of MT
  • All mods: cleanup/refactoring work for MT 5.0.0
  • Add partial translation into Portuguese by BrunoMine (thx!)
  • doc_basics: Translate into German
  • doc_basics: Update help text content for new 5.0.0 goodies
  • doc_items: Better line-breaking
  • doc_items: Add factoid for slippery blocks
  • doc_items: Recognize group fake_liquid=1. If this is present for a block, liquid-related infos are removed to avoid some weirdness. This group is recommended for cobwebs that are internally implemented as liquids.
  • doc_items: Also reveal item entries on inventory interactions


@BrunoMine: Your translation has been included. But because of the new version, some strings are now outdated.
@bell07: Rejected. This is out of scope as the functionality is only useful in combination with a specific crafting guide.
