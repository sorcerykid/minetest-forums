World of Potter
Note: This modpack is not complete. Expect bugs. That said, please report bugs.

World of Potter aims to be a modpack including enough magical elements to make a Dursley smile! You can find it at https://github.com/b3u/world_of_potter

The following mods are ready:
  • Clothes (hpclothes) (kinda)
Currently, the following elements are being worked on:
  • Potions (hppotions)
  • Magical herbs and fungi (hpplants)
  • Wands/spells (hpwands)
  • Misc. decorations (hpdeco)
  • Wizard robes and uniforms (hpclothes)
The following elements are planned:
  • Mobs (both friends and foes)
  • Transportation (or, Apparition)
  • ?
Repo/Download: https://github.com/b3u/world_of_potter
Feature Requests / Bug Reports: https://github.com/b3u/world_of_potter/issues
