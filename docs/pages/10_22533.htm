If you know how to compile your own software, you can download the source code of any version of Minetest and compile it yourself. If you don't know how, search this forum. You might want to place any compiled programs in a location other than where your OS (Ubuntu) installs programs to prevent Ubuntu from deleting or overwriting them. I create a directory named "Programs/Games/Minetest" in my user directory. Good luck.
