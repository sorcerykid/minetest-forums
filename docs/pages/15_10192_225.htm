Re: [Game] Tutorial [2.5.1]

by Wuzzy » Wed Nov 28, 2018 7:43 pm

Actually, there is an Indonesian translation, but it is incomplete. Not everything is translated.

To make sure the translation works, your system language must be set to “Indonesian” (id) before you launch Minetest. Also set the Minetest language to Indonesian.

If you want to help to improve the Indonesian translation, I will gladly merge your work, no matter if it's partial or complete.

Here's how it works:
In each mod, there is a folder named “locale”. You can translate the file named “id.txt”.
It works like this:
Most lines have an English orignal text and an equals sign. Simply write the translation after the equals sign.
If a line starts with “#”, it is a comment for you and does not need to be changed.
Always make sure to test your translation, to make sure that nothing is broken.

If you need help, you can always talk to me in IRC: #minetest in irc.freenode.net.

But keep in mind that it is possible I will change the texts in future versions, especially when there will be new features in Minetest so I have to write new texts.
