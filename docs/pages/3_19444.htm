Re: Hiding player name mod

by AccidentallyRhine » Tue Jan 16, 2018 9:52 pm

Not a mod but setting player_transfer_distance in minetest.conf will do what you want.

Code: Select all
# Whether players are shown to clients without any range limit
player_transfer_distance = 8


I recommend 8 if you are using standard view distance in Minetest.
